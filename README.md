Projekt demonstruje implementację hybrydowej infrastruktury tożsamości w chmurze Microsoft Azure. Celem jest zbudowanie bezpiecznego środowiska opartego na modelu Zero Trust, integrującego lokalne Active Directory (AD DS) z Microsoft Entra ID.

Główne funkcjonalności:

  🛠 Implementacja hybrydowej tożsamości (Entra Connect). <br>
  🛠 Konfiguracja polityk dostępu warunkowego (Conditional Access & MFA).<br>
  🛠 Zarządzanie dostępem przez role (RBAC).<br>
  🛠 Automatyzacja infrastruktury (Infrastructure as Code - Terraform).<br>
